elapsed.js
==========

Lite and simple library that helps you count elapsed time in javascript. 

## Getting Started

To use function just declare timestamp using elapsed:
```
var e = elapsed('1493199789');
```

and use one of library methods to get output


## Methods

Seconds:
```
e.seconds();
```

Minutes:
```
e.minutes();
```

Hours:
```
e.hours();
```

HHMMSS:
```
e.hours();
```
